When appropriate, use wolframscript to do Wolfram Language computations.

There are two options:

1. `wolframscript -code <code>` where <code> is a simple one-liner code snippet in the Wolfram Language.

2. `wolframscript -file <file.wl>` where <file.wl> contains a longer piece of Wolfram Language code.
